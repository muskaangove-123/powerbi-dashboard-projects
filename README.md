# powerbi-dashboard-projects


Introduction :

The purpose of this project is to analyze supermarket sales data and generate meaningful business insights using Power BI. The report helps understand sales performance, customer behavior, and product trends to support better decision-making.


Objective of the Project :

To analyze overall sales performance of the supermarket
To identify top-selling products and categories
To study customer purchase patterns
To evaluate revenue contribution by branches and payment modes
To create an interactive Power BI dashboard


Dataset Description :

The dataset contains transactional data from a supermarket, including:

Invoice ID
Branch
City
Customer Type
Gender
Product Line
Unit Price
Quantity
Tax
Total Sales
Date & Time
Payment Method
Gross Income


Data Preparation : 

The following steps were performed in Power BI (Power Query Editor):

Removed null and duplicate values
Corrected data types (Date, Time, Currency)
Created calculated columns and measures
Extracted Month and Year from Date
Created KPIs
Data Preparation


Key Performance Indicators (KPIs) :

Total Sales Revenue
Total Quantity Sold
Average Customer Rating
Gross Income
Number of Transactions
These KPIs provide a quick overview of business performance.


Slicers are used for:
